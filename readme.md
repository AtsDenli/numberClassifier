This is a neural network, built completely from scratch using cupy (numpy for GPUs)

On this repository, I have written the code to create, optimise and train a neural network for the MNIST digits dataset - 60000 training images and 10000 testing images.
While some parts of the code have been written with this goal weighing on the design decisions, it would be quite easy to adapt it to most situations.

The mathematical parts of the book are derived from a book - Building Neural Networks from Scratch by sentdex. It was a very helpful book and I learned a lot about the 
inner functions of a neural network from it. The non mathematical components - more specifically the "model" class and main.py were written exclusively by me, without 
any help from this book. 

This project is still in development, and more functionality will be added in the coming days
